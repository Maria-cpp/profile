# Assets Folder

This folder is used to store static assets for the website.

## Structure

- `images/` - Store your profile pictures, project screenshots, and other images here

## Usage

To use an image in your HTML files, reference it like this:

```html
<img src="assets/images/your-image.jpg" alt="Description">
```

For CSS backgrounds:

```css
background-image: url('../assets/images/your-image.jpg');
```
